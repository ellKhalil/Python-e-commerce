    ```
    cd project-name
    ```
3. Create a virtual environment:
    ```
    python -m venv env
    ```
4. Activate the virtual environment:
    - On Windows:
        ```
        .\env\Scripts\activate

        ```
        source env/bin/activate

    ```
    pip install -r requirements.txt
